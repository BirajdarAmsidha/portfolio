# Amsidha Birajdar - Portfolio Website

This is my personal portfolio website showcasing my skills, projects, certifications, and contact information. Built using HTML, CSS, JavaScript, and designed for a clean, modern, and responsive look.


## Features
- Responsive design suitable for all devices
- Hero section with animated background and social links
- Skills section with interactive cards
- Projects section with detailed descriptions and GitHub links
- Education, Certifications, and Contact sections
- Clean and modern UI design


## Tech Stack
- HTML5, CSS3, JavaScript
- Bootstrap 5
- Font Awesome for icons
